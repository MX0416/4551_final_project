### Project overview:

1. Control a turtlebot with hand gestures through the laptop's webcam using OpenCV and MediaPipe \n
3. Run the ROS2 SLAM toolbox, and visualize with Rviz2 \n
4. A special hand gesture to spawn a second turtlebot at the first turtlebot's starting location \n
5. Autonomously navigate the second turtlebot to the first turtlebox using the SLAM constructed map built by the first turtlebot
